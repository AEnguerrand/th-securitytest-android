# th-securitytest-android
Tsinghua project - Test max possibility for android path lock

## Dependencies

- C++ 14 Compiler
- CMake (>= 3.9)

## Build and run
### Linux or MacOs
```bash
git clone https://github.com/AEnguerrand/th-securitytest-android.git
cd th-securitytest-android
mkdir build
cd build
cmake ..
cmake --build .
./th-securitytest-android
```
